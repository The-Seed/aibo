## 常用命令集合

``` bash
# 查看git版本
git version

# 抓取远程仓库所有分支更新并合并到本地
git pull

# 将所有修改过的工作文件提交暂存区
git add .

# 从暂存区恢复到工作文件
git reset -- .

# 恢复最近一次提交过的状态，即放弃上次提交后的所有本次修改
git reset --hard
```
